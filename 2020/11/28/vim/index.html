Vim 笔记

发表于 2020-11-28 分类于 Linux ， Vim

Vim (Vi IMproved) 是一个 Unix-like 系统下的文本编辑器，可以在 CLI (命令行界面, Command-Line Interface) 下运行。Vim 学习曲线陡峭，但是掌握后可以完全使用键盘编辑文本，超级高效，因为没有鼠标键盘间的切换打断输入的连贯性。

Vim Logo

0 概念

由于是使用纯命令的方式编辑文本，就会涉及到不同的快捷键和输入模式。Vim 有三种模式：命令模式（Command mode）、输入模式（Insert mode）、 底线命令模式（Last line mode）。

  • 命令模式：每个按键会对应某一动作。 ESC 进入
  • 输入模式：普通的文本输入。 i 进入
  • 底线命令模式：命令模式下按下:（英文冒号）就进入了底线命令模式。 ESC : 进入
